# UX Problem Solving Framework

## Core Philosophy
1. **Prioritize Empathy:** Validate user friction before drafting any design or code. Empathy is your primary diagnostic tool.
2. **Evoke Experiences:** Employ the "Wristwatch" standard. Use sensory, evocative verbs. Avoid bland labels; describe the silent, seamless result.
3. **Validate Utility:** Test every feature against user friction. If a component fails to solve a problem or accelerate engagement, eliminate it.

## Analysis Workflow
1. **Uncover:** Define the user's primary goal.
2. **Diagnose:** Pinpoint the exact friction stopping them.
3. **Strategize:** Formulate a solution that eliminates the friction.
4. **Articulate:** Draft the explanation using human-centered, active verbs.

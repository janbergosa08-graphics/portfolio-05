# Agentic Engineering Workflow

## Philosophy
- **Driver:** You (the user) dictate strategy, path, and intent.
- **Tool:** The AI (the engine) provides execution, precision, and velocity.

## Collaboration Rules
1. **Maintain Precision:** Execute the strategy exactly. Propose changes before altering the course.
2. **Optimize Velocity:** Build modular, maintainable code structures to facilitate instant pivots.
3. **Clarify Why:** Document structural changes using real-world performance and scalability standards.
4. **Target Scope:** Isolate tasks to maintain project integrity and minimize token overhead.

# CosmoQ-Style Portfolio Prompt

Use this prompt with Framer AI, Wix Studio, or any AI text-to-website generator:

---

"Create a high-end, professional UI/UX designer portfolio website. The aesthetic should be 'minimalist modern' with sophisticated motion design, inspired by the fluidity and scroll-triggered storytelling of CosmoQ.

Structure the website based on the following narrative flow:

**Hero Section:** A bold, high-impact headline ('Frictionless journeys that convert users.') with a subtle, mouse-following interactive element.

**Expertise:** A clean, grid-based section highlighting professional experience and project impact.

**Workflow:** An interactive, icon-heavy section explaining AI-integrated design efficiency.

**Process:** A horizontal, scroll-triggered narrative showing the progression from 'Research' to 'Final Solution'.

**Performance:** A card-based section displaying project metrics and client testimonials with 'reveal-on-scroll' animations.

**Insights (FAQ):** A clean, minimalist interactive accordion addressing project management and strategic approach.

**Contact:** A prominent, high-converting call-to-action section with minimal friction.

**Design Guidelines:**

- **Typography:** Large, bold sans-serif headers; high readability for body text.
- **Motion:** Use 'scroll-transform' animations where elements fade or shift as the user scrolls.
- **Interaction:** The site should feel like a guided, immersive experience, not a static page.
- **Color Palette:** Monochrome with subtle, high-contrast accents to ensure project imagery remains the focal point.
- **Tone:** Professional, direct, action-oriented, and authoritative."

---

## 3-Step Content Strategy for Each Project

1. **The "Problem" Card:** Start with one sentence: "The challenge was [X], which caused [Y] friction."
2. **The "Process" Slider:** Show 3 distinct steps: Initial Research → AI-Aided Ideation → Final Impact.
3. **The "Result" Badge:** End with a business-focused metric, e.g., "+30% User Retention" or "48-hour prototype delivery."

---

## Curation Rule

Only pick the 3 projects that best demonstrate the Research → AI → Impact workflow. Keep it focused on the narrative.
